OETF Checker

Go to repository

This tool will let you film a short video of increasing light intensities emitted by this screen. It can then be used to determine your camera’s Opto-Electrical Transfer Function. Because this screen will only light up with pure black and white patterns, its colour callibration won’t come into play, so we can be 100% sure that the amount of light increases linearly.

Filming should be done in a pitch-black environment, where the only significant light source is this screen. You can cover the gap between the camera and the screen with a black cloth, or even make a tinfoil cone to reflect as much light from the screen as possible.

Please set this screen’s backlight intensity (sometimes labelled brightness) to its maximum.

Ideally, the camera’s bare sensor should be facing this screen from a close distance, but if your camera has a fixed lens, please focus it towards infinity.

Assuming this is done to better linearise some other footage, the camera’s settings should be the same as they were during the shoot. Ideally the ISO should be the same too, but in some cases you’ll need to increase it for this screen’s brightness to be enough.
